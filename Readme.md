# Cricket-Match-Scoring-and-Tournament-Management-System
It is a real time cricket score management system where every ball is tracked.It handles all the events like wideball, no ball, bye, leg bye, wickets and runs.
It calculates batsman's score and strike rate in real time along with trend lines.
It also has the feature of calculating net run rate and points after the match is over, make it useful for tournament management.
Its most unique feature is the calcuation of player stats throughout the tournament.

# How to use
1. track a match using cricket_simulator.py
2. you can use generate_match_summary.py to view match summary
3. all the stats can be calculated in the folder Player_stats
4. all the stats can be viewed as an Image using print_stats.py. It just needs adjustment of location of stats file on line number

** DONT REMOVE ANY FOLDER AND FILES UNTIL match_stats.py IS RUN **
